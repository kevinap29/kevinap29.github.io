# Tantangan

Sektor energi membutuhkan keseimbangan antara identitas korporat yang profesional dan informasi teknis yang mudah diakses. Kehadiran PT. GJE sebelumnya terbatas, sehingga sulit bagi mitra potensial untuk memahami cakupan penuh dari solusi energi dan riwayat proyek mereka.

# Implementasi Teknis

Saya memilih **Laravel** karena sintaksnya yang elegan dan fitur keamanan yang kuat, yang sangat penting untuk situs web korporat. Frontend dibangun menggunakan **Bootstrap** untuk memastikan pengalaman yang mengutamakan seluler dan responsif di semua perangkat.

### Keputusan Arsitektur Utama:

- **Blade Templating**: Digunakan untuk komponen UI yang modular dan dapat digunakan kembali.
- **Eloquent ORM**: Mengelola portofolio proyek dan daftar layanan secara efisien.
- **Optimasi SEO**: Mengimplementasikan URL yang bersih dan manajemen meta tag untuk meningkatkan visibilitas mesin pencari.

# Fitur Utama

- **Portofolio Dinamis**: Galeri proyek energi yang telah selesai dan dapat dicari.
- **Modul Layanan**: Halaman detail untuk setiap sektor energi (Terbarukan, Minyak & Gas, dll.).
- **Panel Admin Responsif**: Memungkinkan klien untuk memperbarui konten tanpa pengetahuan teknis.

# Hasil Akhir

Situs web baru ini menyebabkan peningkatan signifikan dalam pertanyaan prospek dan menyediakan platform profesional bagi perusahaan untuk mengamankan kontrak internasional baru.
