# The Challenge

As this was a special project for **my wife**, the pressure was on to deliver a truly exceptional digital experience. Marketing administration roles require a high level of organization and attention to detail. The challenge was to create a digital presence that reflects these qualities—professional, efficient, and visually appealing—while ensuring the site remains fast and accessible across all devices.

# Technical Implementation

The project was built using the latest **Svelte 5** ecosystem. By leveraging **SvelteKit**, we achieved a perfect balance between server-side rendering for SEO and client-side transitions for a smooth user experience.

### Key Features:

- **Svelte 5 Runes**: Implemented `$state` and `$derived` for managing UI states like theme switching and navigation.
- **Multilingual Support**: Integrated **Paraglide JS** to allow visitors to toggle between Indonesian and English, expanding her professional reach.
- **Responsive Design**: Used **Tailwind CSS** to build a mobile-first layout that looks premium on both desktops and smartphones.
- **Glassmorphism UI**: Applied modern design trends with subtle blur effects and elegant typography.

# Final Outcome

The portfolio successfully highlights Nurul Fitri's professional journey and core competencies. It serves as a robust platform that demonstrates not only her skills but also her commitment to professionalism through a high-quality digital experience.
