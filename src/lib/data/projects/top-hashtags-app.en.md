# The Challenge
The Instagram algorithm changes constantly, and what worked yesterday might not work today. Influencers and social media managers needed a tool that was both portable (mobile-first) and data-driven to stay ahead of the curve.

# Technical Implementation
By combining the power of **.NET MAUI** with **MudBlazor**, I created a cross-platform experience that delivers native performance with a rich, responsive UI.

### Key Technical Decisions:
- **Hybrid Blazor**: This allowed me to share over 95% of the UI code between the web and mobile versions, drastically reducing development time.
- **MudBlazor Integration**: Leveraged its extensive component library to create a sleek, professional interface that follows modern design patterns.
- **RapidAPI Backend**: The app consumes data from my own custom-built API, ensuring high data quality and low latency.

# Key Features
- **Niche Discovery**: Categorized hashtag lists for over 50 different social media niches.
- **Real-Time Analytics**: Shows the popularity and competition level for each hashtag.
- **One-Tap Copy**: Streamlined workflow to quickly copy selected tags to the clipboard.

# Final Outcome
The Top Hashtags App has helped hundreds of users optimize their social media strategy, with many reporting a significant increase in their organic reach and follower engagement.
