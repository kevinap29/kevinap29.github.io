# The Challenge
PT. Ciptapapan Dinamika required a centralized system that could handle large volumes of transactional data while remaining user-friendly for staff across different technical backgrounds. Data consistency and real-time reporting were paramount.

# Technical Implementation
The system was built on the **.NET** framework to ensure enterprise-grade stability and performance. **SQL Server** was chosen as the database engine for its robust handling of relational data and complex queries.

### Key Architectural Decisions:
- **Monolithic to Modular**: Structured the code to allow independent updates to different modules (Inventory, HR, Finance).
- **Stored Procedures**: Used extensively to optimize data-heavy operations and ensure data integrity at the database level.
- **Responsive Web UI**: Built with Bootstrap to allow managers to access reports on-the-go.

# Final Outcome
The ERP system reduced administrative overhead by 40% and eliminated 95% of data entry errors, providing management with accurate, real-time insights into company performance.
