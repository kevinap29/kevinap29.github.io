---
title: 'Mempublikasikan Pustaka Utilitas SvelteKit'
---

Setelah membangun beberapa aplikasi SvelteKit, saya menyadari bahwa saya menulis ulang fungsi pembantu dan komponen UI yang sama. Saya mengkonsolidasikannya ke dalam **sveltekit-lib**, sebuah pustaka **TypeScript** _open-source_. Proyek ini mengajarkan saya seluk-beluk memaketkan modul npm, mengonfigurasi Vite/Rollup untuk _build library_, dan mempertahankan definisi tipe data yang ketat.
