---
title: 'Eksplorasi Mendalam tentang Custom HTTP Client'
---

Untuk lebih memahami protokol jaringan dan _interceptor_, saya membangun pustaka _HTTP client_ kustom dengan **TypeScript**. Saya berfokus pada penerapan logika percobaan ulang (_retry_), penyegaran token otomatis, dan pencegat permintaan/respons. Latihan pembelajaran ini memberi saya apresiasi yang jauh lebih dalam terhadap pustaka seperti Axios dan Fetch API bawaan.
