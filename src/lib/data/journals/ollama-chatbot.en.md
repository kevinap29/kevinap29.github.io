---
title: "Local AI Chatbot with Ollama and Svelte"
---

I integrated a local Large Language Model (LLM) into a web interface using **Ollama** and **Svelte**. The challenge was managing the streaming text response smoothly in the UI. By leveraging Sveltes reactivity and the Streams API, I achieved a ChatGPT-like typing effect that runs completely locally without external API costs.
