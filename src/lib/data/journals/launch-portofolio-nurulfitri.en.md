title: 'Launching Portfolio Nurul Fitri'
---

Successfully launched a new portfolio project for **my wife**, Nurul Fitri. This project was built using **SvelteKit** and **Svelte 5**, focusing on a clean, modern, and highly responsive design to showcase her creative work.

Key features include:

- **Internationalization** with Paraglide JS.
- **Modern Styling** using Tailwind CSS.
- **Dynamic Animations** for an engaging user experience.
- **Optimized Performance** for fast loading times.

This project was a great opportunity to explore the latest Svelte 5 features in a real-world production environment.
